Achievements and Notable Events

Website Development Project:

  • Continued enhancements to the HardHat website, focusing on showcasing company projects and improving user experience.
  • Developed features such as user profile management, login authentication, and a resource tracker for project management.

Innofest 2024:

  • HardHat Enterprises participated in Innofest 2024, showcasing their projects and achievements, including the ongoing website development and cybersecurity initiatives.

AppAttack Project:

  • Conducted a comprehensive security analysis of ThotTech's OnTrack Project, including secure code reviews and penetration testing.
  • Delivered a detailed security report with findings and recommendations, contributing to the improved security posture of the OnTrack application.

Website Development Project:

  • Significant enhancements made to the HardHat website, including the addition of new features such as a blog, resource tracker, and admin portal.
  • Deployed the website on PythonAnywhere and Docker to ensure compatibility with microservice deployment architecture.

Penetration Testing GUI (PT-GUI):

  • Released an updated and refurbished version of the Deakin Detonator Toolkits (DDT), a penetration testing tool, with improved functionalities and new tool integrations.

Smishing Detection Project:

  • Developed a prototype smishing detection app with AI-based algorithms to identify smishing content in SMS messages.
  • Initiated the creation of a database for storing smishing patterns and conducted user testing sessions for feedback.

Deakin Threat Mirror Project:

  • Conducted research on open-source threat intelligence platforms and the challenges faced by SMEs and developing economies in cybersecurity.
  • Released the first version of the Deakin Threat Mirror platform, aimed at providing robust cybersecurity solutions for SMEs and developing economies.

Deakin CyberSafe VR Project:

  • Developed interactive VR modules focusing on cybersecurity training for small businesses.
  • Released a Minimum Viable Product (MVP) for the VR platform, including modules on password security, data encryption, and safe web browsing.

Innofest 2024 Continuation:

  • Participation in ongoing events, including demonstrations of the projects developed during T1 and the progress made in T2.

Awards and Recognition:

  • Recognition and awards received for the successful deployment and functionality of the HardHat website and the Deakin Detonator Toolkits (DDT) during industry events.
